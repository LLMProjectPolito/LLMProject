# Complete Bibliography with Verified Links

This document provides academic references with direct links for each of the 11 agent architectures.

## 1. Baseline (Zero/Few-Shot)
- **Paper:** *Language Models are Few-Shot Learners* (Brown et al., 2020)
- **Link:** [https://arxiv.org/abs/2005.14165](https://arxiv.org/abs/2005.14165)

## 2. Actor-Critic (CodeRL)
- **Paper:** *CodeRL: Collaborative Training of Decoding and Rescoring for Software-Program Synthesis* (Le et al., ICLR 2023)
- **Link:** [https://arxiv.org/abs/2207.01780](https://arxiv.org/abs/2207.01780)
- **Justification:** Implements a framework where an actor generates code and a critic (rescorer) evaluates it, mirroring our multi-agent feedback loop.

## 3. Adversarial (AdverTest)
- **Paper:** *Test vs Mutant: Adversarial LLM Agents for Robust Unit Test Generation* (Zhang et al., 2026)
- **Link:** [https://arxiv.org/abs/2602.08146](https://arxiv.org/abs/2602.08146)

## 4. Self-Healing
- **Paper:** *A New Era in Software Security: Towards Self-Healing Software via Large Language Models and Formal Verification* (Charalambous et al., 2023)
- **Link:** [https://arxiv.org/abs/2305.14752](https://arxiv.org/abs/2305.14752)

## 5. Consensus (Self-Consistency)
- **Paper:** *Self-Consistency Improves Chain of Thought Reasoning in Language Models* (Wang et al., 2022)
